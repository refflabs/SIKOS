# 🎨 SIKOS Frontend

This is the modern and responsive client application for the **SIKOS** platform. It provides the user interface for booking management, real-time chat, and admin dashboard operations.

## 🛠️ Technology Stack
* **Framework:** React.js (built with Vite)
* **Styling:** Vanilla CSS with custom typography
* **State Management & Data Fetching:** React Query / Axios
* **Icons:** Lucide React

## 🚀 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Environment Variables
Create a `.env` file in this directory:
```env
VITE_API_URL=/api
```

### 3. Start Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```
